# क्या तुम HACK हो चुके हो? — Viral Hack Checker 🔴

**CodeBites26** presents an interactive cybersecurity awareness quiz to check if your device has been compromised.

## 🎯 Overview

A fun yet informative Hindi-English bilingual web application that helps users understand common cybersecurity vulnerabilities through a 10-question quiz. Get an instant risk score (0-100%) and detailed recommendations.

## ✨ Features

- **10-Question Quiz** - Quick vulnerability assessment
- **Real-time Risk Scoring** - Instant results with percentage scores
- **Detailed Findings** - Specific vulnerabilities identified with explanations
- **Share Functionality** - Share results on WhatsApp and other platforms
- **Bilingual Content** - Hindi-English mix for Indian audience
- **Cyber UI Theme** - Matrix rain animation and hacker aesthetic
- **Responsive Design** - Mobile-first, works on all devices
- **No Backend Required** - 100% static frontend

## 🚀 Quick Start

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/hackcheck.git
   cd hackcheck
   ```

2. Open in browser:
   - Simply open `codebites26_viral_hack_checker.html` in your web browser
   - Or use a local server:
     ```bash
     npx http-server
     # or
     python -m http.server 8000
     ```

3. Visit `http://localhost:8000` (or your server port)

## 📦 Deployment

### Deploy to Vercel (Recommended)
1. Push to GitHub:
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. Connect to Vercel:
   - Go to [vercel.com](https://vercel.com)
   - Import from GitHub
   - Select this repository
   - Deploy!

### Deploy to GitHub Pages
1. Enable GitHub Pages in Settings
2. Select `main` branch as source
3. Your site will be available at `https://yourusername.github.io/hackcheck`

### Deploy to Netlify
1. Drag & drop the HTML file to [netlify.com](https://netlify.com)
2. Or connect your GitHub repository

## 🎨 Risk Levels

- **0-20%**: ✅ SAFE ZONE - Good digital hygiene
- **21-45%**: ⚠️ CAUTION - Fix vulnerabilities soon
- **46-70%**: 🔥 HIGH RISK - Immediate action needed
- **71-100%**: ☠️ CRITICAL - Your device may be compromised
